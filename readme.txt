== Description ==

Plugin baseado em Shopping-Cart integrado com o sistema Pagamento MoIP http://www.moip.com.br. Integra carrinho de compras no menu lateral de wordpress.

1-Com funcionalidade no menu lateral, post ou paginas

2-Possibilita inserir de varios produtos de uma vez

3-Trabalha com integrado diretamente  com  nextgen-gallery com codigo html botao moip diretamente em:

Texto Alternativo (Alt e Title) / Descrito na  URL de seu wordpress com instalando o plugin:

URL plugin Gallery: http://wordpress.org/extend/plugins/nextgen-gallery/

4-Possibilita codigo de integrado na mesma pagina ou post do produto; coloando codigo na pagina

Exemplo: <!--ativar_carrinho_MoIP--> O carrinho apenas sera visivel quando o comprador adicionar pelo menos um 

produto ao carrinho. Aarecendo  no menu lateral se habilitado o Widgest Carrinho e na postagem ou pagina se 

utilizado.

== Installation ==

1-Efetue  upload ou ftp da pasta wordpress-carrinho-moip para o diretorio /wp-content/plugins/.

2-Crie uma conta gratuita para E-Commerce na URL: https://www.moip.com.br/CreateWallet.do

3-Ative o plugin em area administrativa Wordpress > configuracoes > Moip Pagamentos

configura o correio eletronico utilizado como e-mail principal utilizado no sistema MoIP. 

4-Como integrar a seu Wordpress:

4.1 O Primeiro passo sera criar uma conta free no site de www.moip.com.br acessando URL: 

http://www.moip.com.br/MainMenu.do?method=login

4.2 O e-mail configurado na area de configuracoes WordpRess em Moip Pagamentos deve ser o

 e-mail principal de sua conta Moip criada.

4.3- Para adicionar um botao 'Adicionar ao Carrinho' simplesmente insira o texto na pagina ou post  

[moip=Camisa-verde-(M):valor=45.00:end] ao artigo ou pagina, proximo ao produto ou local de sua  escolha.

Substitua NOME-DO-PRODUTO e VALOR-DO-PRODUTO pelo nome e valor real, assim:  

Exemplo: [moip=Camisa-Azul-Tamanho30a53-(M):valor=85.00:end]

 
4.4- Para adicionar o carrinho de compras a um artigo ou pagina de checkout ou sidebar simplesmente adicione o

texto <!--ativar_carrinho_MoIP--> a um post, pagina ou sidebar, caso nao tenha windegst disponivel em seu site sera

possivel incluir o carrinho MoIP em qualquer post que desejar.

Exemplo: <!--ativar_carrinho_MoIP-->

5.Administracao:Plugin para integrado com carrinho de compras para usuarios do sistema Moip.com.br

Utilizando o codigo de produtos Wordpress:

6- Suporta Idioma langs Portugues Brasil pt_BR.PO, Espanhaol Espanha es_ES.PO,Ingles U.S.A us_US.PO
   Edite os arquivo em programa Poedit Versao-1.4.3
